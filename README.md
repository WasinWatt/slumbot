# Slum bot

This is a Slumclub bot implemented using LINE API.
The purpose is to help controlling the messages and announcements in the club.

### Prerequisite

```
go get github.com/line/line-bot-sdk-go/linebot
```

### Install

```
cd slumbot
go install -v
```

### Usage

```
slumbot
```
or

```
cd slumbot
go run main.go
```
